# react-app-setup

> Basic React app setup with Webpack 3, Redux, Redux-thunk, React-router 4, font-generator, scss, post-css, jest

## Install

```sh
$ npm i && npm t
```

## Running

Run application:

```sh
$ npm start
```

Open in browser [http://localhost:8080/](http://localhost:8080/)

## Running tests

Install dependencies and run test

```sh
$ npm t
```
